如何設定?</br>
1.於config.json填入資料</br>
> owner_id填你的id</br>
> prefix填你的前綴</br>
> token就填token</br>
</br>

2.執行 `bot.py`

</br></br></br>

---
指令大全
---

`help`  __幫助列表__
</br>

`start <通知訊息發送頻道id(若為0將會發送到預設的頻道)> <多少訊息上升一等>` __設定群組個性化__
</br>

`oc` __開啟/關閉功能(預設為關閉,第一次設定會自動開啟)__
</br>

`me` __看你的等級和訊息量__
</br>

`xp <id>` __看別人的資料__
</br>

`set <id>` __將指定id等級紀錄歸零__
</br>

`up` __檢查版本__
</br></br>
